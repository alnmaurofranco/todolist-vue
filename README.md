# **Projeto TodoList com VueJS**

## **Bibliotecas e Ferramentas**
    VueJS
    Vuex
    VueRouter
    SpectreCSS
    Eslint

## **Setup**

```
yarn install
```

### **Compila e faz hot-reloads no modo desenvolvimento**
```
yarn serve
```

### **Compila e minifica os arquivos no modo produção**
```
yarn build
```

### **Bugs Lints and fixes files**
```
yarn lint
```
